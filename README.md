# Basic Calculator

A simple Java console application that performs basic arithmetic operations.  
This project was created as part of an internship learning exercise to practice Java fundamentals, input handling, and expression evaluation.

---

##  Features
- Addition (`+`)
- Subtraction (`-`)
- Multiplication (`*`)
- Division (`/`) with division-by-zero handling
- Full expression evaluation (e.g. `5 + 3 * 2 - (4 / 2)`)
- Input validation (no crashes on invalid input)
- Interactive loop — keeps running until the user types `exit`
